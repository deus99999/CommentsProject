# CommentsProject

Full-stack веб-приложение для управления комментариями. Проект построен на архитектуре разделения Backend (Django REST Framework) и Frontend (React + Vite) и полностью контейнеризирован с помощью Docker.

## Технологический стек

- **Backend:** Python 3.11, Django 5.x, DRF, SQLite
- **Frontend:** React 18, Vite
- **Infrastructure:** Docker, Docker Compose.

Функциональные возможности проекта
Проект представляет собой SPA-приложение (Single Page Application) для управления древовидными комментариями с поддержкой мультимедиа и защиты от спама.

📝 Работа с комментариями
Древовидная структура: Поддержка вложенных ответов (Threads). Пользователи могут отвечать на конкретные сообщения.
Валидация данных: * Проверка корректности Email.
Имя пользователя может содержать только латинские буквы и цифры.

Сортировка: Возможность сортировки комментариев по:
Имени пользователя.
E-mail.
Дате добавления (по умолчанию — самые новые сверху).

🛠 Текстовый редактор и HTML
Поддержка HTML-тегов: Система разрешает использование безопасных тегов. Все теги автоматически фильтруются для предотвращения XSS-атак.

Инструментарий: Панель быстрого добавления тегов над формой ввода.

Предпросмотр (Preview): Функция мгновенного просмотра комментария перед отправкой (без перезагрузки страницы).

📁 Мультимедиа и файлы
Прикрепление файлов: Поддержка форматов изображений (JPG, PNG, GIF) и текстовых файлов (TXT).

Автоматическая обработка изображений: * Изображения, превышающие размер 320x240px, автоматически пропорционально уменьшаются на стороне клиента (Image Compression).

Ограничения для TXT: Текстовые файлы проходят проверку на размер (не более 100 КБ).

Просмотр изображений: Интеграция библиотеки Lightbox для удобного просмотра картинок при клике.

🛡 Безопасность и Технологии
Защита от ботов (CAPTCHA): Интеграция графической капчи. Код генерируется на бэкенде, проверяется при каждой отправке формы.

Контейнеризация: Полная поддержка Docker и docker-compose для быстрого развертывания проекта в любой среде.

## Как скачать и запустить проект

Для запуска вам понадобится установленный Docker 

### 1. Клонирование репозитория

Откройте терминал (PowerShell или Git Bash) и выполните:
```bash
git clone https://github.com/deus99999/CommentsProject.git
```
```
cd CommentsProject
```


### 3. Запуск проекта

Соберите и запустите контейнеры одной командой:

```bash
docker compose up --build
```

### Ссылка на проект

Локально проект работает на [http://localhost:3000](http://localhost:3000)
